## What is RichMD?
RichMD is a tool for making Markdown richer.

![RichMD](./docs/images/preview.png)

## Installation

```bash
# use npm
$ npm install richmd

# use yarn
$ yarn add richmd
```

## Markdown Syntax
Please read a [RichMD Markdown Syntax Documentation](./docs/md-syntax.md).

### Suport syntax
- strong
- italic
- image
- link
- headings
- horizontal rule
- blockquote
- unordeed list
- ordered list
- strikethrough
- code block
- checkbox list
- table
- TeX syntax(Use [KaTeX](https://katex.org/))
- Color Inline Block
- Dropdown details
- Import CSS file(Only RichMD file is supported)
- Custom HTML Tag(Only RichMD file is supported)
- Video(HTML5 Video Tag)
### To be implemented in a future release
- Slide view mode
